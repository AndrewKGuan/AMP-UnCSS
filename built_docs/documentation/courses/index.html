Learn Web Development with AMP

These three free courses are in use in schools and training programs around the world. Now you can take them here, online, on our site. Suitable for beginners and experienced web developers alike, they will take you from zero to AMP!

Beginning Course

Learn about AMP and how it helps. Create your first AMP page!

Intermediate Course

Advanced Course

Learn to create advanced user interactions and use server content.

About the courses

Beginning web developers will learn to create modern websites with AMP. Experienced developers will discover how to use AMP to create beautiful, interactive sites while writing less JavaScript. During the course, students build a lovely site for Chico's Cheese Bicycles.

Requirements

  • Internet connection
  • Chrome browser, ideally, to use the AMP Validator extension
  • We use the online editor Glitch. No IDE or local web server needed!

Prerequisites

  • basic knowledge of HTML and CSS
  • for the advanced course, JavaScript expressions and simple JSON
