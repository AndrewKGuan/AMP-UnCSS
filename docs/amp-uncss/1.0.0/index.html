AMP-UnCSS is a way to tree-shake unused CSS from complete AMP HTML documents. It is our attempt at alleviating the work required to manually clean up CSS styles in order to meet the AMP HTML 50KB CSS limit.

Installation Instructions

For use with Webpack

  1. Install via NPM

$ npm install --save amp-uncss

  1. Configure Webpack to include AMP-UnCSS plugin

main/TierOneOps.js

Optimizer that handles CSS optimizations requiring a DOM parser, including: - Removal of all class- and attribute-less selectors that have no corresponding DOM elements - Removal of all selectors (including classes and attributes) that don't have a corresponding DOM element. Only instantiated and used if UnCss config is 1 or higher
